* **Styling:** Uses pure CSS for custom styling of buttons, links, and the overall layout.

## üöÄ Getting Started

To view and interact with the project locally, follow these steps.

### Prerequisites

You need a modern web browser (like Chrome, Firefox, or Edge) and a text editor or IDE.

### Installation and Viewing

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/YOUR_USERNAME/Amazon-Homepage-Clone-HTML-CSS-JS.git](https://github.com/YOUR_USERNAME/Amazon-Homepage-Clone-HTML-CSS-JS.git)
    cd Amazon-Homepage-Clone-HTML-CSS-JS
    ```
2.  **Access the Code:**
    * **IMPORTANT:** To properly view and modify the code, you should open the project folder directly with an IDE like **VS Code** or another compatible code editor.
    * Ensure the **`Amazon.html`** file and the **`images`** folder are present together in the same directory. **The product images and icons are stored in the `images` folder, and the HTML file must be able to access them for the page to display correctly.**
3.  **Run the application:**
    * Simply double-click the **`Amazon.html`** file in your file explorer. It will open automatically in your default web browser.
